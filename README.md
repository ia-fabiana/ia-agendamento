# IA.AGENDAMENTO 🥂✨

**Concierge Digital de Luxo para o Salão da Fabiana**

A IA.AGENDAMENTO é uma plataforma de concierge de altíssimo padrão, desenvolvida para gerenciar agendamentos com elegância, precisão e minimalismo. Utilizando a potência do Google Gemini, ela transforma conversas em experiências de luxo.

## 🚀 Tecnologias
- **Frontend**: React 19 + Vite
- **Estilização**: Tailwind CSS 4 (Design Minimalista/Luxury)
- **IA**: Google Gemini 1.5 Flash (via @google/genai)
- **Animações**: Framer Motion
- **Ícones**: Lucide React

## 🛠️ Configuração

### Pré-requisitos
- Node.js 18+
- Uma chave de API do Google AI Studio (Gemini API)

### Instalação
1. Clone o repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/ia-agendamento.git
   cd ia-agendamento
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Configure as variáveis de ambiente:
   Crie um arquivo `.env` na raiz do projeto:
   ```env
   GEMINI_API_KEY=sua_chave_aqui
   ```

4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
